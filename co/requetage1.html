Présentation

Un des atouts essentiels de la base de données est de pouvoir fournir rapidement des renseignements et informations synthétiques. Romain Rocher compte sur vous pour pouvoir obtenir les éléments suivants:

• Pour travailler, le secrétaire a besoin régulièrement de la liste des adhérents (nom, prénom, numéro, adresse complète) dans l'ordre alphabétique du nom ;

• Les adhérents de la Loire (numéro, nom, prénom, ville) classés par ville ;

• les jeunes (U1 à U18) dans l'ordre décroissant des catégories (n'afficher que les nom, prénom et code de catégorie).

Dans la base de données, créez et enregistrez sous un nom significatif les requêtes correspondantes.
